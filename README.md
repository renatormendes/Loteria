# 🎰 LottoGen Pro - Mr. Magic Programmin 2025

**LottoGen Pro** é um sistema avançado desenvolvido em Java para geração de combinações inteligentes para loterias brasileiras (Mega-Sena, Quina, LotoFácil e Sena).

## 🚀 Funcionalidades
- **Splash Screen Premium**: Carregamento visual com barra de progresso.
- **Lógica Matemática**: Combinações baseadas em operações aritméticas cruzadas de números base (ex: 3, 5 e 7).
- **Estatísticas em Tempo Real**: Painel lateral que exibe a frequência dos números gerados.
- **Persistência de Dados**: Salva e lê jogos em arquivos binários (`.bin`) com timestamp e nome do usuário.
- **Exportação**: Opção para simular impressão e gerar relatórios de jogos.

## 🛠️ Tecnologias Utilizadas
- **Linguagem**: Java 
- **Interface**: Swing (GUI)
- **Persistência**: Serialização de Objetos (Arquivos Binários)
- **Versionamento**: Git
